# Project Context & AI Assistant Instructions

## For Future AI Assistants

### Project Background
This is a sophisticated automated trading bot for the Indian stock market, specifically designed for swing trading strategies. The project has evolved through multiple phases of development and optimization.

### Key Project Files to Review First
1. **CHAT_HISTORY.md** - Contains conversation history and decisions
2. **BULLETPROOF_SYSTEM.md** - Architecture and system design
3. **README.md** - Project overview and setup instructions
4. **config.yaml** - Configuration settings
5. **main.py** & **sniper_swing.py** - Core trading logic

### Development Philosophy
- **Security First**: All credentials are encrypted and securely managed
- **Bulletproof Design**: System handles failures gracefully
- **Modular Architecture**: Clean separation of concerns
- **Comprehensive Testing**: Extensive validation before deployment
- **Real-time Monitoring**: Health checks and notifications

### Important Context
- Uses Zerodha Kite API for trading
- Telegram bot for user interaction and notifications
- Advanced technical analysis with multiple indicators
- AI-powered decision making
- Automated position management
- Risk management with stop-losses and targets
